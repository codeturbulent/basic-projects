# Rupnagar Dashboard (Main)

A modern and responsive dashboard for Rupnagar, Uttar Pradesh, providing essential real-time information for residents and visitors.

## Features
- **Real-Time Weather:** Fetches live weather data from OpenWeather API for the local area (currently configured for Bewar/Rupnagar).
- **Transport Information:** Static and dynamic (planned) train schedules for the Farrukhabad-Rupnagar area.
- **Interactive Map:** Embedded Google Map of Rupnagar and its surroundings.
- **News Section:** A dedicated area for local news (loading placeholder included).
- **Responsive Navigation:** Quick access to Living Here, Things to Do, Gallery, and Development sections.

## Technologies Used
- **HTML5 & CSS3:** (Using custom variables and Flexbox)
- **JavaScript (Async/Await & Fetch API):** For real-time weather and data retrieval.
- **OpenWeather API:** For current weather details.
- **Google Maps Embed API:** For the location view.
- **Google Fonts:** (Oregano)

## How to Use
1. Open `main/index.html`.
2. Browse the dashboard for live weather updates and city information.
3. The weather data is currently set to fetch for the "Bewar" city area (configurable in `script.js`).
