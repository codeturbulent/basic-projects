# Happy Birthday Surprise! 🎂

An interactive and animated birthday surprise web application built with modern web technologies.

## Features
- **Interactive Countdown:** A beautiful loader to build anticipation.
- **Animated Cake:** A virtual cake where you can "light" the candle.
- **Balloon Pop Game:** Pop balloons to reveal a hidden message.
- **Photo Memories:** A cards-effect swiper to display sweet moments.
- **Greeting Card:** A realistic 3D-effect card that opens on tap.
- **Gift Reveal:** A final gift box that opens with a confetti burst.
- **Immersive Effects:** Floating hearts, confetti bursts, and smooth GSAP animations.

## Technologies Used
- **HTML5 & CSS3**
- **Tailwind CSS:** For modern, responsive styling.
- **GSAP (GreenSock Animation Platform):** For smooth and complex animations.
- **Canvas Confetti:** For the celebration effects.
- **Lucide Icons:** For clean, consistent iconography.
- **Swiper.js:** For the interactive photo gallery.
- **Google Fonts:** (Pacifico and Poppins)

## How to Run
Simply open `index.html` in any modern web browser.
