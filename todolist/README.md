# Simple To-Do List

A lightweight, browser-based to-do list application to help you stay organized.

## Features
- Add tasks with a title and detailed description.
- Automatically records the time and date when a task is added.
- Track how many tasks you've completed.
- Remove individual tasks or clear the entire list.
- "Check All" functionality to mark all current tasks as done.
- Responsive table display.

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## How to Use
1. Open `index.html`.
2. Enter a task title and description.
3. Click "Add to list" or press Enter.
4. Hover over a task title to see its full description.
5. Click "Check" when a task is finished to increase your "Tasks Done" count.
6. Use "Remove" to delete a task without marking it as done.
