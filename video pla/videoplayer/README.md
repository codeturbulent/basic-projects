# Custom Video Player

A feature-rich, interactive video player built from scratch using HTML5, CSS3, and Vanilla JavaScript.

## Features
- **Custom Controls:** Play/Pause, Volume Control, and Progress Bar.
- **Speed Options:** Adjustable playback speed (0.5x to 2x).
- **Video Timeline:** Hover to preview time and click/drag to seek.
- **Picture-in-Picture:** Switch to PiP mode with a single click.
- **Full Screen:** Toggle full-screen mode.
- **Keyboard Friendly:** Use backward/forward buttons to skip 5 seconds.
- **Auto-hide Controls:** Controls fade out after 3 seconds of inactivity.

## Technologies Used
- **HTML5 Video API**
- **CSS3 (Custom styling and icons)**
- **JavaScript (Event listeners, DOM Manipulation)**
- **FontAwesome & Google Material Icons**

## How to Use
1. Open `index.html`.
2. Replace `housefull5a.mp4` in `index.html` with your own video file name to play your content.
3. Use the control bar to interact with the video.
