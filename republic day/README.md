# Republic Day Parade Watcher

A dedicated landing page to watch the Indian Republic Day parade and learn more about this national festival.

## Features
- **Live Stream:** Embedded YouTube player to watch the parade.
- **Interactive Feedback:** Like and Dislike buttons to share your sentiment (connected to a Google Apps Script).
- **Share:** WhatsApp share integration to invite friends.
- **Donate:** Quick link for donations.
- **Educational Content:** Information about the history and significance of Republic Day.

## Technologies Used
- HTML5
- CSS3
- YouTube Embedded Player API
- Google Apps Script (Backend for feedback)

## How to Use
1. Open `index.html`.
2. Interact with the video player to watch the parade.
3. Use the like/dislike buttons to vote (one-time use per session).
4. Share the link with friends via the WhatsApp button.
