# UDMVI Personal Portal

A personal dashboard for accessing files, uploading content, and playing music.

## Features
- **File Access:** Quick link to Google Drive folders.
- **Upload Utility:** Link to a Google Apps Script for file uploads.
- **Music Player:** A dedicated `song.html` page that acts as a directory for local music files (Note: requires a `songs/` folder with music to function).
- **Responsive Layout:** Grid-based interface for easy navigation.

## Technologies Used
- HTML5
- CSS3 (Flexbox)
- JavaScript (for search functionality in the music player)

## How to Use
1. Open `index.html` to access the main dashboard.
2. Navigate to "Files", "Upload", or "VMs" (Music Player).
3. The "download my songs" link currently points to a local `songs/` directory.
