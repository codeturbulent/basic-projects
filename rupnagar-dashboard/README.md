# Rupnagar Project

This repository contains two versions of a municipal website for Rupnagar.

## Contents
- **[main/](./main/):** The primary Rupnagar Dashboard project, featuring real-time weather, news, maps, and transport information.
- **Root [index.html](./index.html):** A comprehensive municipal template (based on the iCreate platform) with advanced UI components and assets.

## Technologies Used
- HTML5
- CSS3
- JavaScript (jQuery)
