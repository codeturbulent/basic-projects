# Simple Web Calculator

A clean and functional calculator built with HTML, CSS, and JavaScript.

## Features
- Basic arithmetic operations (Addition, Subtraction, Multiplication, Division).
- Clear screen functionality.
- Responsive grid layout.
- Error handling for invalid expressions.
- Dual-screen display (history and current input).

## Technologies Used
- HTML5
- CSS3 (Grid layout)
- JavaScript (eval-based calculation)

## How to Use
1. Open `index.html` in your browser.
2. Click the buttons to input your calculation.
3. Click `=` to see the result.
4. Click `C` to clear the screen.
