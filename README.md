## 📂 Project Categories

### 🎮 Games & Interactive Experiences
*   **[8-Number Puzzle](./8number/)**: A logic-driven grid game requiring strategic placement of numbers 1-8. Features custom adjacency validation.
*   **[Birthday Surprise](./birthday/)**: An immersive celebration app featuring GSAP animations, interactive cake lighting, and a virtual gift reveal.

### 🛠️ Utilities & Tools
*   **[DarkSide Encoder](./DarkSide/)**: A secure text-to-file encoder using randomized base conversions (radix 10-36) and secret keys embedded in filenames.
*   **[JS Calculator](./calculator/)**: A sleek, grid-based arithmetic calculator with error handling and calculation history.
*   **[To-Do List](./todolist/)**: A task manager with dynamic time-stamping, task completion tracking, and "Check All" functionality.
*   **[Contact Viewer](./contacts/)**: A client-side file reader using the HTML5 FileReader API to preview text-based contact files.
*   **[New Year Countdown](./newyear/)**: A real-time countdown timer designed with CSS blending modes and dynamic intervals.

### 📊 Dashboards & Portals
*   **[Rupnagar Dashboard](./rupnagar/)**: A city-specific municipal portal integrating real-time weather via OpenWeather API, Google Maps, and local transport schedules.
*   **[Personal Portal (UDMVI)](./UDMVI/)**: A centralized hub for personal file management, uploads, and a custom music directory.

### 📱 Media & UI Clones
*   **[Custom Video Player](./video/)**: A high-performance HTML5 video player with playback speed control, Picture-in-Picture, and custom UI skin.
*   **[Instagram Clone](./instagram/)**: A pixel-perfect recreation of the Instagram login interface, featuring a responsive layout and backend form integration.
*   **[Republic Day Watcher](./republic/)**: A multimedia landing page for national events featuring embedded streaming and interactive feedback loops.

## 🛠️ Technical Stack
Across these projects, you'll find implementation of:
- **Core:** Vanilla JavaScript (ES6+), HTML5, CSS3.
- **Layout:** CSS Flexbox & Grid, Media Queries for Responsive Design.
- **APIs:** Fetch API, OpenWeather API, YouTube Data API, Google Maps Embed.
- **Animations:** GSAP (GreenSock), CSS Keyframes.
- **Libraries:** Tailwind CSS, Swiper.js, jQuery (in municipal templates).

## 🚀 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/codeturbulent/basic-projects.git
    ```
2.  **Run any project:**
    Navigate to any folder and open the `index.html` file in your preferred web browser. No server or build step is required for most of these projects.
