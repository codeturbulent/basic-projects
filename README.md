# Web Development Projects Collection

A collection of various mini-projects and utilities built with HTML, CSS, and JavaScript.

## 📁 Projects Overview

### 1. [8-Number Puzzle](./8-number-puzzle/)
A logic puzzle game where you arrange numbers 1-8 in a grid, following strict adjacency rules.
- **Key Features:** Real-time rule validation, unique number checking.
- **Tech:** HTML5, CSS3, JavaScript.

### 2. [Birthday Surprise](./birthday-surprise/)
An interactive and animated birthday celebration with a countdown, interactive cake, and balloon pop reveal.
- **Key Features:** GSAP animations, Tailwind CSS, Canvas Confetti.
- **Tech:** Tailwind CSS, GSAP, Swiper.js.

### 3. [JS Calculator](./js-calculator/)
A clean and functional web-based calculator for basic arithmetic.
- **Key Features:** Expression history, responsive grid UI.
- **Tech:** HTML Grid, JavaScript (eval).

### 4. [Contact File Viewer](./contact-viewer/)
A simple utility to read and display the content of text files locally.
- **Key Features:** Client-side File API integration.
- **Tech:** JavaScript (FileReader).

### 5. [Text Encoder/Decoder](./text-encoder-decoder/)
A security-focused tool to encode sensitive messages into text files using randomized bases.
- **Key Features:** Randomized base encoding, file download/upload support.
- **Tech:** JavaScript Blob API, Custom Logic.

### 6. [New Year Countdown](./new-year-countdown/)
A beautiful countdown timer for the upcoming New Year celebration.
- **Key Features:** Live timer, background image blending.
- **Tech:** JavaScript (setInterval).

### 7. [Republic Day Parade Watcher](./republic-day-parade/)
A dedicated landing page to watch the parade and learn about India's history.
- **Key Features:** YouTube Embed API, Google Apps Script feedback loop.
- **Tech:** HTML5, CSS, YouTube API.

### 8. [Rupnagar Dashboard](./rupnagar-dashboard/)
A municipal portal featuring real-time weather, maps, and transportation schedules.
- **Key Features:** OpenWeather API integration, responsive municipal UI.
- **Tech:** JavaScript (Fetch API), jQuery.

### 9. [To-Do List](./todo-list-js/)
A lightweight task manager with time-stamping and completion tracking.
- **Key Features:** Dynamic task management, local task counter.
- **Tech:** JavaScript DOM Manipulation.

### 10. [Personal Portal (UDMVI)](./personal-portal-udmvi/)
A central dashboard for personal file management and music streaming.
- **Key Features:** File upload/access links, custom music player directory.
- **Tech:** Flexbox UI, JavaScript.

### 11. [Custom Video Player](./custom-video-player/)
A feature-rich video player with custom controls and advanced playback options.
- **Key Features:** Playback speed control, Picture-in-Picture, auto-hide UI.
- **Tech:** HTML5 Video API, Material Icons.

---
## 🚀 How to Run
Most projects are pure client-side and can be run by simply opening the `index.html` file in any modern web browser.
