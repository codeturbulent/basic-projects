# Web Development Projects Collection

A collection of various mini-projects and utilities built with HTML, CSS, and JavaScript.

## 📁 Projects Overview

### 1. [8number](./8number/)
A logic puzzle game where you arrange numbers 1-8 in a grid.

### 2. [birthday](./birthday/)
An interactive and animated birthday celebration surprise.

### 3. [calculator](./calculator/)
A clean and functional web-based calculator.

### 4. [contacts](./contacts/)
A simple utility to read and display the content of text files locally.

### 5. [DarkSide](./DarkSide/)
A security-focused tool to encode and decode sensitive messages.

### 6. [instagram](./instagram/)
A pixel-perfect clone of the Instagram login page with a backend integration.

### 7. [newyear](./newyear/)
A beautiful countdown timer for the upcoming New Year celebration.

### 8. [republic](./republic/)
A dedicated landing page to watch the parade and learn about India's history.

### 9. [rupnagar](./rupnagar/)
A municipal portal featuring real-time weather, maps, and transportation.

### 10. [todolist](./todolist/)
A lightweight task manager with time-stamping.

### 11. [UDMVI](./UDMVI/)
A central dashboard for personal file management and music.

### 12. [video](./video/)
A feature-rich video player with custom controls.

---
## 🚀 How to Run
Open the `index.html` file in each folder to view the project.
