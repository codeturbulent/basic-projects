# Contact File Viewer

A simple utility to read and display the content of text files locally using the browser's File API.

## Features
- Select and upload any text-based file.
- View the file content directly in the browser.
- No server-side processing required (fully client-side).

## How to Use
1. Open `index.html`.
2. Click "Choose File" and select a text file (e.g., .txt, .csv, .json).
3. Click "View Content" to see the file's data.

## Technologies Used
- HTML5
- JavaScript (File API / FileReader)
