# Instagram Login Clone

A pixel-perfect clone of the Instagram login page, demonstrating responsive UI design and form handling.

## Features
- **Responsive UI:** Adapts to different screen sizes, hiding the illustrative image on smaller devices.
- **Form Handling:** Captures login credentials and sends them to a Google Apps Script backend.
- **Realistic Effects:** Includes an initial "Logged out" alert and redirects to a specific Instagram reel after a "login" attempt.
- **Modern Styling:** Uses Flexbox for layout and mimics Instagram's official aesthetic.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (Fetch API, DOM Manipulation)
- Google Apps Script (Backend for form submission)

## How to Use
1. Open `index.html`.
2. Observe the responsive layout as you resize the browser window.
3. Attempting to log in will trigger a data submission to the configured script URL and redirect you to a sample Instagram post.
