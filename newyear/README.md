# New Year Countdown

A simple and elegant countdown timer for the upcoming New Year.

## Features
- Real-time countdown (Days, Hours, Minutes, Seconds).
- Beautiful background image with a blended overlay effect.
- Responsive design with central alignment.
- Custom typography using Google Fonts (Vujahday Script).

## Technologies Used
- HTML5
- CSS3
- JavaScript (setInterval)

## How to Use
1. Open `index.html`.
2. The countdown will automatically start based on your current local time.
